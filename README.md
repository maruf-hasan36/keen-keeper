## 🧠 Overview

**KeenKeeper** is a modern, elegant friendship management platform that helps you stay intentionally connected with your important people.

It allows you to track interactions (calls, texts, video calls), visualize your engagement over time, and never let a meaningful relationship fade due to a busy schedule.

Built with **Next.js**, it delivers a fast, responsive and delightful user experience across all devices.

### 🏠 Smart Dynamic Homepage

- Personalized "Your Friends" section
- AI-inspired data mapping and insights
- Real-time rendering from structured data

### 🎴 Interactive Friend Cards

- Rich profile details: name, tags, goals, contact info
- Quick actions:
  - 📞 Call
  - 💬 Text
  - 🎥 Video Call
- Instant toast notifications for every action

### 🕒 Advanced Interaction Timeline

- Complete history of all interactions
- Filter by: Calls, Texts, Video Calls
- Powerful search by friend's name

### 📊 Beautiful Data Visualization

- Built with **Recharts**
- Track engagement trends over time
- Color-coded analytics for Calls, Messages, and Video interactions

### 📱 Fully Responsive Design

- Optimized for **Mobile**, **Tablet**, and **Desktop**
- Consistent and polished UI across all screen sizes

### ⚡ Performance & UX

- Fast navigation with Next.js App Router
- Loading spinners for better perceived performance
- Smooth animations and minimal design
- Toast notifications using React Toastify
